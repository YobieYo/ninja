# Ночь ниндзя LAN

Браузерная многопользовательская игра в жанре социальной дедукции по мотивам «Ночи ниндзя».  
Запускается как локальный веб-сервер — все игроки подключаются с телефонов или браузеров в одной сети.

## Роли

| Роль | Кол-во | Способность |
|------|--------|-------------|
| 🥷 Ниндзя | ~1/3 от игроков | Каждую ночь вместе выбирают жертву. Видят друг друга. |
| ⚔️ Самурай | Остальные | Нет ночных действий. Голосует днём. |
| 🔍 Сэнсэй | 1 (при 5+ игроках) | Каждую ночь узнаёт, является ли выбранный игрок ниндзя. |
| 💚 Целитель | 1 (при 7+ игроках) | Каждую ночь защищает одного игрока от убийства. |

## Ход игры

1. **Ночь** — все «закрывают глаза». Каждая роль совершает своё действие через экран своего устройства.
2. **День** — объявляется, кто погиб. Сэнсэй видит результат своей проверки. Все голосуют, кого изгнать.
3. Игра продолжается до победы одной из сторон.

**Победа самураев** — все ниндзя уничтожены.  
**Победа ниндзя** — ниндзя равны по числу самураям или превышают их.

## Запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сервер

```bash
python app.py
```

### 3. Откройте игру

- На этом же устройстве: `http://localhost:8000`
- На других устройствах в сети: `http://<IP-адрес-хоста>:8000`

> Узнать свой IP-адрес: `ipconfig` (Windows) или `ip a` / `ifconfig` (Linux/macOS).

### 4. Играйте

- Первый вошедший игрок становится **ведущим** (👑).
- Ведущий нажимает **«Начать игру»** когда все подключились.
- Каждый игрок видит свою роль и действует со своего устройства.
- После окончания партии ведущий может начать новую.

## Технический стек

- **Backend**: [FastAPI](https://fastapi.tiangolo.com/) + WebSocket (asyncio)
- **Frontend**: Vanilla HTML/CSS/JS (без зависимостей)
- **Синхронизация**: WebSocket push-уведомления всем клиентам

## Примечания

- Состояние игры хранится в памяти. Перезапуск сервера сбрасывает партию.
- Все роли скрыты; ниндзя видят друг друга, остальные — нет.
- На время разработки в `app.py` включён `reload=True`.
